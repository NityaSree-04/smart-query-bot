# How to Get FREE Gemini API Key

## Step 1: Go to Google AI Studio
URL: https://aistudio.google.com/app/apikey

## Step 2: Sign In
- Use your Google account (same one with Gemini Pro)
- Accept terms if prompted

## Step 3: Create API Key
1. Click "Create API Key" button
2. Select "Create API key in new project" (or use existing project)
3. Your API key will be generated immediately!
4. Copy the key (starts with "AIza...")

## Step 4: Add to Your Project
1. Open: C:\Users\nitya\OneDrive\Desktop\Ideathon\api\config.php
2. Find line with OPENAI_API_KEY
3. Replace with:
   define('GEMINI_API_KEY', 'AIza-your-key-here');

## Benefits of Gemini API
